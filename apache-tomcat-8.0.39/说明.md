只用 javac（不用ant，maven，gradle）批量编译.java（包括子文件夹下的）的方法：

```
# Linux
$ find -name "*.java" > sources.txt
$ javac @sources.txt

:: Windows
> dir /s /B *.java > sources.txt
> javac @sources.txt
```
